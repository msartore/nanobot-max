# Skills

The following skills extend your capabilities. Each skill is a script you invoke via the `exec` tool.

To use a skill:
1. Read its SKILL.md using `read_file` on the `<location>` path to learn the exact command syntax.
2. Call `exec` with the command shown in the SKILL.md, substituting `{baseDir}` with the `<baseDir>` value from the listing below.

Skills with available="false" need dependencies installed first.

**Important**: Skills are NOT native functions — always invoke them via `exec` with the shell command from their SKILL.md.
